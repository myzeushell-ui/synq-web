# Synq Roadmap

## Stage 1 — Investor Demo ✅ (current)
**Goal**: Show the product vision to investors without a backend.

- [x] Next.js web demo with mock data
- [x] Full mobile Flutter app (Android, local persistence)
- [x] Home screen, Thoughts tab, Emotions tracker, Reminders
- [x] Capture modal with auto-category detection
- [x] Task states: active / done / paused / overwhelmed
- [x] Calm deadline copy and emotional safe language
- [x] Insight cards on home screen
- [x] Onboarding flow
- [x] Demo mode banner

---

## Stage 2 — MVP (1–2 months)
**Goal**: Real users can install the Flutter app and use it daily.

- [ ] Android APK via Play Store (internal testing)
- [ ] iOS build (TestFlight)
- [ ] Local notification reminders (flutter_local_notifications)
- [ ] Offline-first persistence upgrade (SQLite via drift)
- [ ] Category lock + split preview improvements
- [ ] Settings screen (data export, theme toggle)
- [ ] Onboarding A/B test

---

## Stage 3 — AI Mock Experience (2–3 months)
**Goal**: Demonstrate AI emotional intelligence without a live model.

- [ ] Pre-scripted "Talk to Synq" conversation flows
- [ ] Pattern analysis: "You feel anxious before deadlines" insights
- [ ] Mood trend graph (7-day / 30-day)
- [ ] RSD support flow: "This feels like rejection — that's okay."
- [ ] Contextual suggestion engine (rule-based, no API)

---

## Stage 4 — Backend Foundation (3–5 months)
**Goal**: Persistent data, multi-device sync, user accounts.

- [ ] Supabase (PostgreSQL + auth + realtime)
- [ ] User account creation (email / Google OAuth)
- [ ] Cross-device sync
- [ ] Data encryption at rest
- [ ] GDPR-compliant data export and deletion

---

## Stage 5 — AI Integration (5–8 months)
**Goal**: Real AI-powered emotional support.

- [ ] Claude API integration (emotional analysis, suggestions)
- [ ] Therapist-reviewed CBT prompt library
- [ ] Adaptive pacing: Synq learns your patterns
- [ ] Crisis detection + safe referral (if distress detected)
- [ ] Weekly emotional summary email

---

## Stage 6 — Mobile First-Class (6–10 months)
**Goal**: Production-ready mobile apps.

- [ ] iOS App Store submission
- [ ] Android Play Store public release
- [ ] Push notifications (FCM)
- [ ] Widget (today's tasks, quick capture)
- [ ] Apple Watch / Wear OS companion

---

## Stage 7 — Growth & Monetisation
**Goal**: Sustainable business model.

- [ ] Free tier: unlimited capture, 7-day history
- [ ] Pro tier ($8/mo): AI insights, unlimited history, reminders, export
- [ ] Team tier ($25/mo): shared capture spaces, team emotional check-ins
- [ ] B2B: employee wellbeing tool for HR teams
