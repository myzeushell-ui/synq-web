# Synq — Emotional-safe thought capture

> **Investor Demo** — open the app and explore without login or backend.

Synq is a mobile-first app that helps people capture thoughts, emotions, tasks, and reminders — without judgment. Built for humans with busy minds, anxiety, or ADHD.

---

## 🚀 Quick Start

```bash
git clone https://github.com/your-org/synq-web.git
cd synq-web
npm install
npm run dev
# → http://localhost:3000
```

## 🌐 Deploy to Vercel

Connect GitHub repo at vercel.com → Import → synq-web. No env vars required for demo.
